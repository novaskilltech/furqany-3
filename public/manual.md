# Mode d'emploi de l'application

Bienvenue sur votre application d'apprentissage ! Voici comment l'utiliser :

## 1. Configuration du profil
Lors de votre première connexion, il vous sera demandé de renseigner votre prénom et votre âge. Ces informations permettent à Lumi, votre assistant, de personnaliser ses messages pour vous.

## 2. Assistant Lumi
Lumi est votre guide. Il vous accompagne tout au long de votre apprentissage.
- Cliquez sur le bouton **"Lumi, explique-moi ce verset ! ✨"** pour obtenir une explication adaptée aux enfants.
- Lumi s'adressera à vous par votre prénom.

## 3. Navigation et apprentissage
- Explorez les différents quartiers pour découvrir de nouveaux versets.
- Certains quartiers peuvent être protégés par un code parental.

## 4. Paramètres
Vous pouvez personnaliser l'apparence de l'application via le menu des paramètres.
